## What You Will Learn

This project covers essential Spring Core concepts and practical usage. You will learn:

- Basics of Spring Framework and its architecture  
- Dependency Injection (DI) and Inversion of Control (IoC) container  
- Spring Beans, their lifecycle, and configuration  
- Constructor Injection and Setter Injection  
- Injection of Collection types (List, Set, Map, Properties)  
- Resolving ambiguity in bean injection  
- Lifecycle methods of Spring Beans  
- Implementing bean lifecycle using interfaces (`InitializingBean` and `DisposableBean`)  
- Autowiring using XML configuration or annotations  
- Standalone collections and usage with Spring's util schema  
- Stereotype annotations (`@Component`, `@Service`, `@Repository`, `@Controller`)  
- Spring Bean scopes (singleton, prototype, etc.)  
- Spring Expression Language (SpEL) for dynamic values  
- Using boolean types and expressions with SpEL
